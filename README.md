**设计亮点说明**
1. 左侧导航 (IA架构)：清晰地将功能区分为“Dashboard（概览）”、“AI Studio（生产工具）”和“Resources（资产）”，符合SaaS用户心智。
2. Dashboard：采用了“快捷入口卡片”设计，让用户一进来就能点击核心功能，减少跳出率。右上角高亮显示“余额”，时刻进行商业化暗示。
3. 视频翻译页 (Video Translator)：
- Side-by-Side 视图：左边视频，右边脚本。
- 脚本编辑器：这是我们产品差异化的核心。普通的翻译软件是黑盒，我们允许用户改脚本，强调“可控性”。
4. 数字人制作页 (Avatar Studio)：
- Excel 风格：摒弃了传统的表单，直接用表格布局。这暗示了“批量生产”的能力，用户一看就知道能处理大量SKU。
5. 商品生成页 (Product Shot)：
- 场景卡片：用具体的风格（高级展台、户外自然）代替复杂的Prompt输入框，降低了用户的使用门槛。
